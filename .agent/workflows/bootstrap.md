# Bootstrap Agent Workflow

## Context

Generate `AGENTS.md` (or `GEMINI.md`) to define the Agent's Framework.

## Prerequisites

- [ ] Project directory exists.

## Steps

1. **Run Init Script**

    ```bash
    python3 dev_ops/commands/init_project.py
    ```

2. **Verify Setup**
    - Ensure `AGENTS.md` is created and configured.

## Exit Criteria

- [ ] `AGENTS.md` exists in root.
