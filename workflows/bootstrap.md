# Bootstrap Workflow

## Steps

1. Run `python3 scripts/bootstrap.py`.
2. The script will detect your project's coding languages.
3. It will prompt you for configuration values (e.g., versions, libraries).
4. Provide the requested values to customize your rules.
5. Rules will be installed to `.agent/rules/`.

## Exit Criteria

- [ ] `.agent/rules/` is populated with customized rules.
