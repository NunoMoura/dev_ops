# Create README Workflow

## Steps

1. Create a `README.md` file.
2. Populate it with the template below.

## Template

```markdown
# {{project_name}}

> [!IMPORTANT]
> **ACTION REQUIRED**: Provide a clear overview of the project.

## Overview

[Description]

## Installation

[Installation steps]

## Usage

[Usage examples]

## Development

[Dev instructions]
```
