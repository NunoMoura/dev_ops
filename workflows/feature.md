# Create Backlog Workflow

## Prerequisites

- [ ] Project is initialized.

## Steps

1. Create a `dev_docs/backlog.md` file.
2. Populate it with the template below.

## Template

````text
```yaml
type: backlog
status: active
```

# Backlog

> [!IMPORTANT]
> **ACTION REQUIRED**: Maintain a prioritized list of tasks and features.

## High Priority

- [ ] Task 1

## Medium Priority

- [ ] Task 2

## Low Priority

- [ ] Task 3
````

## Exit Criteria

- [ ] `docs/backlog.md` file created.
- [ ] Initial items added to appropriate priority sections.
